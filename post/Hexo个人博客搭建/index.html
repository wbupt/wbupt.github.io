Hexo个人博客搭建

2023-03-14

title: Hexo个人博客搭建

  1. 安装nodejs、git

  2. 在文件夹中右击 Git Bash Here

  3. 使用npm命令安装Hexo，输入：

    npm install -g hexo-cli
hexo init
hexo new 我的新博客
hexo g #生成
hexo s #启动服务预览
hexo d #部署
# 打开浏览器输入地址
localhost:4000
hexo clean #清除缓存，若是网页正常情况下可以忽略这条命令

  4. 配置网站

进入根目录里的themes文件夹，里面也有个_config.yml文件

deploy:
  type: git
  repo: git@github.com:仓库名/仓库名.github.io.git
  branch: main
  • 安装Git部署插件
npm install hexo-deployer-git --save # 安装上传文件
hexo clean 
hexo g 
hexo d

ssh-keygen -t rsa -f  ~/.ssh/wbupt -C "2934785957@qq.com"
ssh -T [2934785957@qq.com protected]
ssh -T git@wbupt.github.com #新的ssh_key验证

(git新设置）分别进入两个Hexo博客.git目录下执行以下命令单独设置用户名/邮箱

git config user.name "beiyoudaxue"
git config user.email "wxyfyz@163.com"
